# [Sentinel](https://hash.vercel.app)
- Uptime Management platform for critical monitoring for each deployments.
- Curated Dashboard for each and every statistics.
- Email reminder for critical downtimes.

## Getting Started with Development

1. Clone the repo:
```sh
git clone https://github.com/Hyperdunks/ump
```
2. Copy the `.env.example` to `.env` and Put credentials in `.env`:
```sh
cp .env.example .env
```
3. Run the dev server:
```sh
bun run dev
```
